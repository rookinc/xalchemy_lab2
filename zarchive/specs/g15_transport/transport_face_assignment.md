# G15 transport face assignment

This file assigns faces for the transport system using the tetrahedron as the primitive object.

## Core claim

Faces are tetra-native.

Chirality is transported.

The shell is a projection.

## Reading rule

The tetrahedron is read as:

- three manifest faces
- one closure face

This matches the observed transport reading:

- three faces become legible first
- the fourth face folds in
- tetrahedral closure is then completed

## Primitive faces

The primitive tetra faces are:

- `F0`
- `F1`
- `F2`
- `F3`

## Assigned roles

- `F0` = left manifest face
- `F1` = right manifest face
- `F2` = base/front manifest face
- `F3` = closure face

So the tetra is not introduced as four equally presented faces at once.

Instead, the transport lens presents:

- `F0, F1, F2` as the manifest shell
- `F3` as the face that folds in to complete closure

## Adjacency

Tetra face adjacency is primitive.

Each face meets the other three:

- `F0` touches `F1, F2, F3`
- `F1` touches `F0, F2, F3`
- `F2` touches `F0, F1, F3`
- `F3` touches `F0, F1, F2`

## Witness reading

In the front-face tetra camera, the observer stands outside the tetra and sees the manifest shell first.

That means the system is read as:

1. open trihedral shell
2. interior transport core
3. folded closure by `F3`

## Oriented face states

Raw face labels are not yet chirality labels.

The oriented lifted states are:

- `F0+`, `F0-`
- `F1+`, `F1-`
- `F2+`, `F2-`
- `F3+`, `F3-`

The `+/-` distinction is carried by lifted transport through sheet memory.

So:

- face identity is tetra-native
- orientation is transport-retained

## Chirality

Chirality does not live in the raw face names.

Chirality emerges when transport retains sheet memory and the same face can appear in opposite oriented states.

That happens first at the lifted layer, not at bare face assignment.

So the strongest closure reading is:

- the tetra gives the faces
- the fold gives the closure role
- the lift gives the chirality

## Projection to shell

The shell regions:

- upstairs
- downstairs
- stairs

are not primitive tetra faces.

They are projected regions of oriented face transport.

So the shell is useful and real as a lens, but it is not the ontological root.

## Short form

- three manifest faces plus one closure face
- `F3` folds in
- faces are tetra-native
- oriented faces live at the lift layer
- shell regions are projected, not primitive
