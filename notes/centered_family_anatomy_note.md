## 5. Weak refinement question

The weak-sensitive operator candidate is:

Qhat_sigma

derived from the signed lift / cocycle layer.

The key question is:

How does Qhat_sigma act on the coarse family decomposition 5 + 4 + 5?

There are three main possibilities:

### Case A: family preservation
Qhat_sigma preserves E18, E3, and E2 as families.

Then:
- the coarse families are already correct blocks
- weak structure only decorates them internally

### Case B: internal splitting
Qhat_sigma preserves each coarse family as a whole but splits one or more families into smaller invariant subfamilies.

This is currently the most interesting possibility.

Examples:
- 5 -> 1 + 4
- 5 -> 2 + 3
- 4 -> 1 + 3
- parity-like + / - classes

If this occurs, these smaller invariant pieces become the first true particle-family candidates.

### Case C: cross-family mixing
Qhat_sigma mixes distinct coarse families.

Then:
- E18, E3, E2 are only ordinary coarse families
- the true particle families must be defined jointly by the operator system generated by:
  - Qhat
  - Qhat_sigma

This is richer but harder.
