# Species Core / Swap Note

## Purpose
This note records the exact decomposition of each doubled theorem row species into:
- fixed core
- partner swap sets
- envelope

## Definitions
For a doubled species with rows R+ and R-:

- core = R+ ∩ R-
- first-only = R+ \ R-
- second-only = R- \ R+
- envelope = R+ ∪ R-

So:
- each row = core plus one side of the swap
- the envelope is the full species chamber range

## Current structural result

### Flank species
F0, F1, F2 have:
- small fixed core
- large envelope
- exact 2-for-2 swaps in each of the five chamber families B0..B4

Interpretation:
flank species are broad mobile selectors controlled by a global blockwise involution.

### Companion species
C0, C1, C2 have:
- larger fixed core
- smaller envelope
- sparse local swaps supported only on species-relevant chamber families

Interpretation:
companion species are tight chamber identities with local partner retuning.

## Working grammar
The theorem row system is generated by:

1. chamber-family decomposition
2. species template
3. partner involution

In symbols:

- rigid anchor selectors: A0, A1, A2
- flank templates with involutions: F0±, F1±, F2±
- companion templates with involutions: C0±, C1±, C2±

This is the current clean theorem-side chamber grammar.
