Show that Qhat_sigma is not zero and is not redundant.

- Qhat_sigma is not a polynomial in Qhat
- Qhat_sigma contributes genuinely new centered structure

This is the first real break point.

Compare the coarse centered decomposition

Need to determine whether Qhat_sigma:

Any nontrivial refinement is enough to establish a first theorem.

### Lemma 4: invariant-subfamily lemma
Once the joint action is known, identify the first proper invariant subfamilies of

A = <Qhat, Qhat_sigma>.

These become the first honest particle-family candidates.

## 6. Failure modes

The theorem target could fail in several ways.

### Failure mode A
Qhat_sigma = 0 after centering

Then the weak layer does not enter the centered arena in this form.

### Failure mode B
Qhat_sigma is present but algebraically redundant with Qhat

Then no new family structure is produced.
The weak layer would not refine the centered families.

### Failure mode C
Qhat_sigma acts, but only trivially on the coarse family decomposition

Then the current particle-family program is too optimistic or the bridge is too coarse.

### Failure mode D
The bridge operator has been defined incorrectly

Then a finer or different lift-sensitive construction is needed.

These are legitimate scientific outcomes.
Any one of them would clarify the real shape of the problem.
